# 腾讯新闻平台写作风格分析

> 基于 3 篇腾讯新闻范文 + 1 篇平台规则页的风格拆解，用于多平台内容改写训练。

---

## 例文 1

**平台**：腾讯新闻
**链接**：https://news.qq.com/rain/a/20260524A034BG00
**标题**："泡药杨梅"被封杀后，果农紧急抛售，收购价30块跌到8毛钱
**内容类型**：深度资讯（社会调查报道）
**作者**：每日人物（企鹅号官方账号）

### 标题特征

- 标题类型：资讯型 + 冲突型混合
- 关键手法：「泡药杨梅」制造争议标签 + 「被封杀」定性事件严重性 + 「30块跌到8毛钱」用极端价格反差制造冲击
- 结构：争议事件定性（泡药杨梅被封杀）+ 后果呈现（果农抛售）+ 数字对比（30块→8毛钱）
- **标题模式**：「[争议标签]被[打击]后，[受害者]紧急[行动]，[价格]从[X]跌到[Y]」

### 开头原文摘录

> "初夏顶流水果"的陨落，只需要几天。
>
> 5月15日，福建电视台的一则曝光，让福建漳州龙海区浮宫镇的杨梅收购点违规使用添加剂、甜蜜素浸泡杨梅的问题浮出水面。5天后，官方通报来了，5家涉事收购点共查扣问题杨梅540公斤，违规添加剂20.1公斤，刑事拘留5人。
>
> 消息传开，收购商一夜之间跑光了。曾经一斤能卖二三十块的漳州杨梅，直线跌到1块多一斤，甚至8毛钱。果农们凌晨四五点上山，骑着摩托车在崎岖山路一趟趟地把果子运下山，在临时支起的收购点前却抢不到一个收购筐。拉下山一车100多斤杨梅，排队两三个小时，只能换来微薄的几十块钱。
>
> 这场风波没有止步于产地。1000多公里外，上海的批发市场销毁了1641公斤不合格杨梅，商超们纷纷紧急标注"不泡药"、附带检测报告，就连茶饮品牌们也连夜标明杨梅产地，福建杨梅一夜间从市面上消失了……

### 开头特征总结

**先给结论 + 逐层放大冲击感**。第一句用"陨落"定调，一句话制造悬念；第二段用精确时间线和具体数据（540公斤、20.1公斤、5人）还原事件；第三段切换到果农视角，用极端价格对比（20-30块→8毛钱）和场景细节（"抢不到一个收购筐"）拉满共情；第四段从产地扩散到全国终端市场，形成"产地→上海→全网"的三级放大。整个开头在4段内完成「定性→数据→共情→扩散」的四连击，每段都在扩大故事的冲击半径。

### 中段原文摘录

> **杨梅熟了，却卖不出去了**
>
> 凌晨五点，天还没亮，陈可馨就已经在山上摘杨梅了。她是请假回来的。父亲在电话里说："今年请不起工人了，你能不能回来帮帮忙？"
>
> 往年这个时候，村里总是热闹的。摩托车、小车、货车把路堵得水泄不通，收购商的吆喝声此起彼伏……但今年不一样了。父亲把摩托车骑到收购点，看到的是一群人围着一辆货车，争抢塑料筐。谁抢到了，才能把自己的杨梅倒进去、称重、卖掉。抢不到的，只能载着杨梅继续往几十公里外的下一个收购点碰运气。
>
> **信任坍塌之后**
>
> 但信任已然破裂。几乎没人愿意相信，果农对此毫不知情。消息传出后，整个产区的杨梅都被贴上了"泡药"的标签。最快跑走的也正是村子里的收购商。刘峰那天照常摘了杨梅拉下山，发现常去的收购点门关了，再跑一家，也关了……
>
> 风波不仅击穿了产地，也迅速蔓延到终端市场。西郊国际农产品交易中心……销毁了检测不合格的杨梅1641公斤。……目前杨梅日均批发量已从四五十吨跌至不足10吨，环比下降约80%。

### 中段特征总结

**人物故事 + 数据支撑 + 多线并行**。中段用黑体小标题（"杨梅熟了，却卖不出去了""信任坍塌之后""一棵杨梅树，养了几代人"）划分为多个章节。每个章节采用「个体故事开道 → 普遍现状补充 → 行业/数据佐证」的三层结构：先用陈可馨、刘峰、木木等具体农户的故事建立代入感，然后切换到宏观数据（日均批发量从50吨跌至10吨、环比下降80%），再从产地延伸到茶饮品牌、线上平台、历史危机等多条线索。故事线和数据线交替推进，避免读者疲劳。

### 结尾原文摘录

> 700年前，浮宫人开始在山上种杨梅，700年后，他们的子孙还在做同一件事。不是因为他们没有别的选择，而是因为这片山，只认杨梅。
>
> 而杨梅又是"风吹一半，雨落全无"的果子。只是今年的这场"风雨"，比往年残忍了许多。

### 结尾特征总结

**时空拉伸 + 意象收束**。用"700年前→700年后"的长时间轴做历史纵深，把眼前的危机拉入一个更宏大的叙事框架——这不是一次偶发的灾难，而是小农经济世代循环中的一个节点。最后用"风吹一半，雨落全无"这句农谚做意象收束，把"风雨"双关为自然的风雨和行业的风波，余味悠长。不说教、不煽情、不呼吁行动，用一句轻描淡写留下沉重的余韵——这正是腾讯新闻深度报道的典型收法。

### 语言风格判断

- **段落长度**：中短（3-5句为主，偶有单句强调段，人物对话短段频繁）
- **语气**：冷静克制但充满人文关怀，用事实和细节说话，不直接评论，情感藏在故事里
- **信息密度**：极高（大量精确数字、企业/机构名称、时间节点、地理信息、人物引语）
- **平台感强不强**：极强，典型腾讯新闻深度调查报道风格——"用脚采访，用笔还原"
- **主风格**：资讯调查（60%）+ 故事叙述（25%）+ 行业分析（15%）

### 最值得学习

1. **标题** — 争议标签 + 极端价格反差，一句话讲完一个完整的事件逻辑
2. **开头结构** — "定性→数据→共情→扩散"四段式，每段扩大冲击半径，节奏极快
3. **中段编排** — 个体故事与宏观数据交替推进，"一个人→一个村→一个产业"的缩放镜头
4. **结尾** — 不用评论收尾，用历史纵深和农谚意象代替观点，让读者自己感受

### 风格标签

`数据密集` `人文关怀` `多线叙事` `小标题分段` `空间缩放` `故事驱动` `引语丰富` `冷静克制` `纵深收束` `第三方视角`

## 例文 2

**平台**：腾讯新闻
**链接**：https://news.qq.com/rain/a/20260525A029MO00
**标题**：《阿嬷》破10亿！清华教授：观众重新拿回评价权
**内容类型**：观点评论（行业分析）
**作者**：文娱春秋（企鹅号娱乐领域创作者）

### 标题特征

- 标题类型：资讯型 + 观点型
- 关键手法：热点电影（《阿嬷》）+ 里程碑数据（破10亿）+ 权威背书（清华教授）+ 观点提炼（观众重新拿回评价权）
- 结构：事件成果 + 冒号分隔 + 权威人士核心观点，用"重新拿回"暗示之前评价权被剥夺，制造认知张力
- **标题模式**：「[作品/事件]破[里程碑数字]！[权威身份]：[争议性观点]」

### 开头原文摘录

> 5月中旬，远在法国南部海滨的戛纳国际电影节举办之际，国内影市也因为一部电影热闹起来——蓝鸿春执导的《给阿嫲的情书》。
>
> 这部没有参赛戛纳、甚至此前没在其他电影节上拿奖的作品，成为了当下中国电影行业里的"无冕之王"——以1400万的成本，目前斩获了破十亿的票房，如果真如预测所估最终达到18亿，那么，它将成为中国影史最高收益率的电影——超40倍（分账票房/成本）。
>
> 《给阿嫲的情书》的爆，不是大明星路演喊出来的，是在影院里慢慢"烧"起来——先从小范围的观众口碑开始，接着被朋友圈、短视频、坊间评论一点点推高。这使得这部最初看起来"不太像爆款"的电影，变成了如今绕不过去的名字。

### 开头特征总结

**先设场景对照 + 再引爆数据**。第一句用戛纳和国内影市做空间对照，拉出"远方电影节"和"身边票房奇迹"的对比张力；第二句立刻用成本/票房比（1400万→10亿→预测18亿→收益率超40倍）做数据轰炸；第三句用"烧"这个比喻解释爆火机制——不是人工造势，是自然蔓延。三段之间从「哪里热闹→多热闹→为什么这么热闹」逐层递进，一气呵成。

### 中段原文摘录

> **-1-** 差异化、共情性、自来水式的口碑
>
> 所谓差异化，说白了就是观众真的看腻了那些太熟的东西。大明星、大制作、大IP、大宣发，当然还是有用，但它们越来越不能自动换来信任……
>
> 《给阿嫲的情书》的潮汕话不是装饰，侨批也不是随手贴上的文化标签。那些信，那些漂洋过海的钱，那些没法当面说出口的话，构成了一种真实的生活纹理。电影没有急着告诉观众"这是一个多么重要的中国故事"，它只是先把一个家拍实了……
>
> **-2-** 评价权到底发生了什么变化？
>
> 以前一部电影好不好，常常是专家先说，媒体跟上，观众最后买单。现在，顺序反过来了。有些片子是观众先把它捞起来，专家和行业再回头解释它为什么会火。
>
> 尹鸿认为，现在已经不是专家关起门来评电影的时代了——豆瓣、猫眼、短视频、B站、朋友圈，都在直接影响一部电影的命运……

### 中段特征总结

**分段论述 + 现象→本质递进**。用数字序号（-1-/-2-/-3-/-4-）划分为四个章节，形成"电影为什么火→评价权怎么变了→行业该怎么回应→中国电影如何走出去"的递进逻辑链。每个章节核心是"引述权威观点 → 拆解观点含义 → 补充案例佐证"的三段式。引语比例极高（尹鸿、秦振贵、法国影评人等），依托权威发言展开论述，这是腾讯新闻行业评论的典型手法。

### 结尾原文摘录

> 真正的问题是，那些没有爆出来的电影怎么办？
>
> 同样真诚、同样地方、同样小成本的作品，有多少死在前三天排片里？有多少根本没机会被观众看见？有多少创作者甚至等不到被误解的那一天？
>
> 如果每次都要靠观众把一部电影从角落里捞出来，那不是市场成熟，而是机制偷懒。
>
> 所以，《给阿嫲的情书》的意义不只是"爆火"，像一次提醒：行业的触觉并不总是准确。很多真正有生命力的东西，最开始看起来都不够标准、不够安全、不够好卖。

### 结尾特征总结

**反转追问 + 尖锐收束**。在全文围绕"爆火"做了大量正面分析之后，结尾突然翻转——"那些没有爆出来的电影怎么办？"用三个连续反问句（"有多少死在前三天排片里？""有多少根本没机会被观众看见？""有多少创作者等不到被误解的那一天？"）把话题从赞扬拉向反思。最后用"机制偷懒"四个字一针见血收束。不提供答案，只提出问题——这是腾讯新闻评论文章的高阶收法：留白让读者自己思考。

### 语言风格判断

- **段落长度**：短（2-4句为主，观点句常单句成段）
- **语气**：冷静分析 + 尖锐批判，有明确的行业立场但不情绪化，用反问而非感叹
- **信息密度**：高（大量引用专家原话、具体数据、行业术语）
- **平台感强不强**：极强，腾讯新闻行业观察/文娱评论的典型写法——"依托现场发言，做深度拆解"
- **主风格**：观点评论（70%）+ 行业分析（20%）+ 资讯播报（10%）

### 最值得学习

1. **标题** — 热点作品 + 里程碑数字 + 权威金句提炼，三个信息层次叠在一个标题里
2. **开头** — 空间对照 + 数据轰炸 + 传播机制解释，三段讲完"什么事""多大""为什么特别"
3. **中段结构** — 权威引语驱动论述，每个论点都有明确的来源和出处，增强可信度
4. **结尾** — 从正面分析翻转到批判反思，用反问句制造冲击，不提供廉价答案

### 风格标签

`权威引语` `分段递进` `数据轰炸` `现象→本质` `反问收束` `行业视角` `冷静批判` `短段落` `留白结尾` `戛纳现场感`

## 例文 3

**平台**：腾讯新闻
**链接**：https://news.qq.com/rain/a/20260524A09B8300
**标题**："全力以赴做好伤员救治"——山西通洲集团留神峪煤矿瓦斯爆炸事故受伤人员救治见闻
**内容类型**：资讯通讯（官方新闻通稿风格）
**作者**：新华社新闻（企鹅号官方账号）

### 标题特征

- 标题类型：引语型 + 事件型
- 关键手法：直接引用官方表态（"全力以赴做好伤员救治"）作为主标题 + 破折号后补充完整事件信息（地点+事故类型+报道角度）
- 结构：官方引语定调 + 破折号分割 + 事件全称 + 报道体裁（"见闻"暗示一线采访）
- **标题模式**：「[官方表态引语]」——[完整事件名称]+[报道体裁标识]

### 开头原文摘录

> 5月22日19时29分，山西省长治市沁源县山西通洲集团留神峪煤业有限公司井下发生瓦斯爆炸事故，造成重大人员伤亡。当前受伤人员救治情况如何？24日，记者进行了采访。
>
> "目前两名危重症伤者各项指标相对稳定。"刚刚从长治医学院附属和平医院重症监护室走出，省级专家组成员、山西医科大学第一医院重症医学科主任医师郭焱接受采访时说。

### 开头特征总结

**先报事件 + 再给最新进展**。第一句是标准的新闻导语——精确到分钟的时间、完整的机构全称、事件定性（瓦斯爆炸事故）、伤亡概述（重大人员伤亡），所有五要素（When/Where/Who/What/How）在第一句全部交代。第二句用设问"救治情况如何？"自然过渡到核心报道内容。第三句直接引述权威专家最新判断。整个开头干净利落，零铺垫零抒情，是典型的新华社通稿体。

### 中段原文摘录

> 128名受伤人员中，2名危重症伤者最为牵动人心。郭焱表示，从5月23日到24日，在国家、省、市、县四级医疗人员的努力下，两名危重伤者的病情都在好转，其中一名已在24日上午拔除了气管插管，意识状态、血氧饱和度、生命体征等都比较平稳……
>
> 另有两名重症伤者被收治在长治市人民医院。记者24日在医院急诊重症科看到，4名医护人员正围在一名伤者床旁，通过调试呼吸机参数，尝试让伤者尽快脱离呼吸机……
>
> 记者了解到，目前对2名重症伤员和2名危重症伤员实行了"一人一专班、一人一方案"精细化救治，事故124名轻伤员被收治在长治市7家三甲医院中，也在积极开展治疗。

### 中段特征总结

**逐级报道 + 现场见闻穿插**。中段按伤情严重程度分层报道：危重症→重症→轻症，每一层都以"记者看到/记者了解到"的现场观察作为信息源，穿插医护人员直接引语和伤员自述。结构高度标准化：「现场观察句 + 数据/措施 + 当事人引语 + 预后判断」。段落之间用"另有两名""记者了解到""记者24日在"等过渡语串联，信息扎实但不花哨。

### 结尾原文摘录

> 记者了解到，在国家卫健委的指导下，山西省同步启动心理干预服务，派出10名专业人员对伤员开展心理疏导。"经过第一轮摸底干预，轻症伤者整体上较为稳定。"山西医科大学第一医院精神卫生科医生宋洁说。（记者马晓媛、马志异、杨晨光）

### 结尾特征总结

**信息补充 + 署名收束**。结尾不做总结、不抒情、不升华，只是补充了"心理干预"这一新的信息维度后直接收笔。括号内的记者署名是新华社通稿的标志性格式，表明这是一篇标准的一线采访稿件。这种结尾方式追求的是信息完整性而非阅读体验，体现了官方媒体"报道事实、不做评论"的职业规范。

### 语言风格判断

- **段落长度**：中（3-5句为主，每段有明确的单一信息点）
- **语气**：客观、中性、官方，完全避免任何情绪化表达和主观判断，用"记者看到/了解到"替代"令人揪心/值得欣慰"等情感词
- **信息密度**：极高（精确时间、机构全称、人名职务、数字统计，几乎每句都有可核实的具体信息）
- **平台感强不强**：中强，这是新华社通稿的标准格式，放在腾讯新闻上更多体现的是"权威信源"的转载价值，而非腾讯原生风格
- **主风格**：资讯通报（90%）+ 现场见闻（10%）

### 最值得学习

1. **导语写法** — 第一句塞满五要素，不铺垫不渲染，适合硬新闻的快速消费场景
2. **分层报道** — 按"危重症→重症→轻症"逐级展开，结构清晰，读者可以快速跳读到自己关心的信息层级
3. **引语使用** — 每段核心信息都有明确的人物引语来源（姓名+职务+所在机构），增强权威性和可信度

### 风格标签

`导语优先` `五要素完整` `逐级分层` `引语驱动` `零抒情` `官方口吻` `署名收束` `信息完整>阅读体验` `现场观察`

### 学习价值

这篇是企鹅号（腾讯新闻内容开放平台）的官方发文规范页面。它的核心价值在于：

1. **标题规范**：标题应在5-64字范围内，禁止使用特殊符号、emoji、全英文、连续标点、标题党套路（震惊/出大事/删前速看等）。对比上文三篇范文可以发现，腾讯新闻的标题虽然也追求冲击力，但走的是"事实+数据"的硬路线，而非情绪化标题党路线。
2. **内容红线**：严禁谣言、低俗色情、血腥暴力、惊悚恶心、虚假编造等内容。这解释了为什么三篇范文中虽然有煤矿爆炸这样的灾难题材，但没有血腥细节描写——平台规范要求"带有血腥暴力场面的图片请打马赛克或者使用远景避免展示特写及细节"。
3. **低质内容打击**：禁止标题党（情绪类夸张/程度类夸张/绝对化用词/悬念类标题）、机器翻译拼凑、旧闻冒充新闻等。这解释了腾讯新闻上深度报道能获得推荐的原因——平台在机制层面排斥低质内容。
4. **广告规范**：禁止在正文中推广个人联系方式、软文广告、二维码导流等。这保证了腾讯新闻内容的相对纯净度。

简言之，这份规范定义了腾讯新闻的"下限"——它告诉我们在腾讯新闻发布内容时绝对不能做什么，以及平台推荐算法倾向于什么类型的内容。

## 腾讯新闻平台风格总结

| 维度 | 腾讯新闻典型特征 |
|------|------------------|
| **标题** | 事实先行 + 数字驱动 + 避免情绪化标题党，偏好「事件定性+关键数据+影响范围」的三段式结构 |
| **开头** | 1-4段快速切入：时间/地点/事件→数据/细节→影响/扩散，零铺垫，每段都在扩大信息半径 |
| **中段** | 小标题分段（深度报道）或序号分节（评论文章），单段≤5句；深度报道靠人物故事+数据交替推进，评论文章靠权威引语驱动论述 |
| **结尾** | 深度报道用意象/历史纵深收束，不直接评论；评论文章用追问/反思收束，不提供廉价答案；官方通稿用信息补充+署名收束 |
| **段落** | 中短（3-5句为主），人物引语段可短至1-2句，评论观点段也常单句成段 |
| **语气** | 冷静、克制、第三方视角。深度报道用人文关怀代替直接表态，评论文章用逻辑和引语代替情绪化输出 |
| **信息密度** | 极高。精确到分钟的时间、完整的机构全称、具体的数字、带姓名职务的引语——几乎每一段都有可核实的具体信息 |
| **内容类型谱系** | 深度调查（人文关怀+数据密集） > 行业评论（权威引语+逻辑拆解） > 官方通报（五要素完整+零抒情） |
| **核心公式** | 热点事件 + 具体数据/人物故事 + 多维度展开 + 克制的收束 |

### 与知乎的对比速览

| 维度 | 腾讯新闻 | 知乎 |
|------|---------|------|
| **标题** | 事实+数据驱动，少见情绪词 | 结论先行+冲突词，常有反转/悬念 |
| **开头** | 逐层扩大冲击半径（产地→全国） | 场景/情绪定调后快速给出观点 |
| **语气** | 均匀克制，从头到尾保持同一温度 | 先冷静后情绪化，有递进层次 |
| **个人信息** | 极少出现"我/笔者认为"，全程第三方 | 常出现"我/笔者认为"，观点立场鲜明 |
| **结尾** | 意象/追问收束，不留明确结论 | 总结+情绪收束，用排比/警句做句号 |
| **平台调性** | 媒体机构的专业主义 | 个人/专栏作者的观点表达 |
